# Link Preview — Crimson Edition 🚀

A premium, high-performance link previewer built with React and Vite. This edition features a sleek **Red & Gray** design system, high-end typography, and an Apple-inspired dark UI.

See exactly how your links look before sharing them — across every major platform.

---

## ✨ New in This Edition
- **Crimson Design System** — A sophisticated Red & Gray palette with deep charcoal surfaces.
- **Plus Jakarta Sans** — Premium typography for a professional, high-end feel.
- **Glassmorphism UI** — Advanced backdrop blurs and saturated overlays for the top-tier navigation.
- **Enhanced Micro-interactions** — Polished shadows, focus glows, and smooth transitions.

---

## 🚀 Features
- **6 platform previews** — X (Twitter), Facebook, LinkedIn, WhatsApp, Slack, Discord.
- **OG Score** — 0–100 quality score based on present meta tags.
- **OG Tag Inspector** — View all raw Open Graph & Twitter Card meta tags.
- **OG Tag Editor** — Live-edit title, description, and image URL to test changes in real-time.
- **Compare Mode** — Paste two URLs side by side for direct quality comparison.
- **Bulk Mode** — Check multiple URLs at once, one per line.
- **History** — Your last 10 lookups saved locally for instant access.
- **Shareable Links** — Every preview creates a unique `?url=` link.
- **Character Limit Warnings** — Real-time alerts when your content exceeds platform limits.
- **⌘K Shortcut** — Quickly focus the input from anywhere.
- **Light / Dark Mode** — Seamlessly toggle between two beautifully crafted themes.

## 🛠️ Tech Stack
- **Frontend**: React 18 (Vite)
- **Styling**: Vanilla CSS (Custom Design System, Zero Frameworks)
- **Data**: Microlink API (Real-time metadata scraping)
- **Typography**: Plus Jakarta Sans
- **Optimization**: Zero-runtime CSS, Client-side processing

---

## 📦 Installation & Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/selvaganapathycoder/link-preview.git
   ```

2. **Install dependencies**
   ```bash
   cd link-preview
   npm install
   ```

3. **Run locally**
   ```bash
   npm run dev
   ```
